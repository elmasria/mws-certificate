## Manifest Icon size recomendation

Help to ensure icon for 1x, 2x, 3x, and 4x devices

* 48px by 48px
* 96 by 96px
* 128px by 128px
* 144px by 144px
* 192px by 192px
* 256px by 256px
* 384px by 384px
* 512px by 512px
